# Tutorial missions — guided operations

These files are **step-by-step training missions** for humans *and* scaffolding for assistants. Each tutorial uses **copy-paste briefs** with numbered deliverables and explicit phases so the model is less likely to skip strategy, confuse copy with layout, or ship unverified claims.

**Why that matters:** Large language models follow **structure** well. Atomic objectives plus clear “done when” criteria act like a lightweight spec — they deepen reasoning and make debriefs (PASS / PARTIAL / FAIL) possible without inventing criteria after the fact.
